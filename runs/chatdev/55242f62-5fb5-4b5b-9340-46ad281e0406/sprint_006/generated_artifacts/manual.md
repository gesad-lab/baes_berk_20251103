# Course Management System

A comprehensive application for managing courses, students, and teachers, allowing for easy enrollment and relationship management.

## Main Functions

- **Create Students**: Add new students to the system with their name and email.
- **Create Courses**: Add new courses with a name, level, and an optional teacher.
- **Create Teachers**: Add new teachers with their name and email.
- **Enroll Students in Courses**: Enroll students in specific courses.
- **Manage Relationships**: Establish relationships between courses and teachers.

## Installation

To set up the Course Management System, you need to install the required dependencies. Follow these steps:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/your-repo/course-management-system.git
   cd course-management-system
   ```

2. **Create a Virtual Environment** (optional but recommended):
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

3. **Install Dependencies**:
   Use pip to install the required packages:
   ```bash
   pip install fastapi sqlalchemy pydantic alembic
   ```

4. **Set Up the Database**:
   The application uses SQLite by default. You can change the database URL in `database.py` if needed.

5. **Run Migrations**:
   Ensure that the database schema is up to date by running the migration scripts:
   ```bash
   alembic upgrade head
   ```

6. **Start the Application**:
   Run the FastAPI application:
   ```bash
   uvicorn main:app --reload
   ```

## How to Use the Application

Once the application is running, you can interact with it through the API endpoints. Here are some examples of how to use the main functionalities:

### 1. Create a Student
- **Endpoint**: `POST /students/`
- **Request Body**:
  ```json
  {
    "name": "John Doe",
    "email": "john.doe@example.com"
  }
  ```

### 2. Create a Course
- **Endpoint**: `POST /courses/`
- **Request Body**:
  ```json
  {
    "name": "Mathematics 101",
    "level": "Beginner",
    "teacher_id": 1  // Optional
  }
  ```

### 3. Create a Teacher
- **Endpoint**: `POST /teachers/`
- **Request Body**:
  ```json
  {
    "name": "Jane Smith",
    "email": "jane.smith@example.com"
  }
  ```

### 4. Enroll a Student in a Course
- **Endpoint**: `POST /students/{student_id}/enroll/`
- **Request Parameters**:
  - `student_id`: ID of the student to enroll
  - `course_id`: ID of the course to enroll in
- **Example**:
  ```bash
  curl -X POST "http://localhost:8000/students/1/enroll/?course_id=1"
  ```

## Additional Information

For more detailed documentation on the API endpoints and their usage, please refer to the FastAPI documentation or the autogenerated Swagger UI available at `http://localhost:8000/docs` once the application is running.

## Conclusion

This Course Management System provides a robust framework for managing educational entities. With the ability to create and manage students, courses, and teachers, it simplifies the process of educational administration.
