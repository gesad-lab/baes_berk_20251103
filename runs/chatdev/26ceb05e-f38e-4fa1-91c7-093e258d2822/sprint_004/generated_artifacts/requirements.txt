
### Explanation of Dependencies:
- **fastapi**: The web framework used to create the API.
- **uvicorn**: The ASGI server for running the FastAPI application.
- **sqlalchemy**: The ORM used for database interactions.
- **pydantic**: Used for data validation and settings management.
- **requests**: Used for making HTTP requests in the GUI.
- **tkinter**: The GUI toolkit used for the application interface (usually included with Python installations).
