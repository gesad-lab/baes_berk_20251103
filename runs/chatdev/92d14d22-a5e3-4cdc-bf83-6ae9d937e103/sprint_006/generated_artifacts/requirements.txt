
### Explanation of Dependencies:
- **fastapi**: The web framework used for building APIs.
- **uvicorn**: The ASGI server for running FastAPI applications.
- **sqlalchemy**: The SQL toolkit and Object-Relational Mapping (ORM) system used for database interactions.
- **pydantic**: The data validation and settings management library used for defining schemas.
