
### Explanation of Dependencies:
- **fastapi**: The web framework used to create the API.
- **uvicorn**: The ASGI server used to run the FastAPI application.
- **sqlalchemy**: The ORM (Object Relational Mapper) used to interact with the SQLite database.
- **pydantic**: Used for data validation and settings management, particularly for the `BaseModel` and `EmailStr` used in the API.
