
### Explanation of Dependencies:
- **fastapi**: The web framework used for building the API.
- **uvicorn**: The ASGI server for running FastAPI applications.
- **sqlalchemy**: The ORM used for database interactions.
- **alembic**: The migration tool for handling database schema changes.
- **requests**: The library used for making HTTP requests in the GUI.
- **tkinter**: The GUI toolkit, which is included with Python, so no version is specified.
