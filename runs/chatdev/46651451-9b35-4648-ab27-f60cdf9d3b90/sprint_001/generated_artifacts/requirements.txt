
### Explanation of Dependencies:
- **fastapi**: This is the main framework used to build the web application, providing the necessary tools to create APIs quickly and efficiently.
- **uvicorn**: An ASGI server that is used to run the FastAPI application. It is lightweight and designed for high performance.
- **sqlalchemy**: This is the Object Relational Mapper (ORM) used for database interactions, allowing for easy manipulation of database records through Python objects.
- **pydantic**: This library is used for data validation and serialization, ensuring that the data being processed meets the defined schema.
