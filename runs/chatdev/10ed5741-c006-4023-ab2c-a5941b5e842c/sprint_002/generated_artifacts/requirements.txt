
### Explanation of Dependencies:
- **fastapi**: The web framework used to create the API endpoints for the Student entity.
- **uvicorn**: The ASGI server used to run the FastAPI application.
- **sqlalchemy**: The ORM library used for database interactions and defining the Student model.
- **requests**: The library used in the GUI to make HTTP requests to the FastAPI endpoints.
- **tkinter**: The GUI toolkit used for creating the user interface. It is part of the Python standard library, so it does not require a specific version.
