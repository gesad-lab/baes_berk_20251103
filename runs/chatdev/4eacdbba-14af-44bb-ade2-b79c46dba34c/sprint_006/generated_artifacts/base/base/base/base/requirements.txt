
### Explanation of Dependencies:
- **fastapi**: The web framework used for building the API.
- **uvicorn**: The ASGI server for running the FastAPI application.
- **sqlalchemy**: The ORM used for database interactions.
- **pydantic**: Used for data validation and settings management.
- **requests**: For making HTTP requests in the GUI.
- **alembic**: For handling database migrations.

### Note:
- **tkinter**: The GUI toolkit is included with Python's standard library and does not require installation, so it is not listed in the `requirements.txt`.
