
### Explanation of Dependencies:
- **fastapi**: The web framework used to build the API.
- **uvicorn**: The ASGI server for running the FastAPI application.
- **sqlalchemy**: The SQL toolkit and Object-Relational Mapping (ORM) system used for database interactions.
- **alembic**: The database migration tool for SQLAlchemy, used to manage database schema changes.
- **pydantic**: The data validation and settings management library used for defining request and response schemas.
