
### Explanation of Dependencies:
- **fastapi**: The web framework used to create the API.
- **uvicorn**: An ASGI server for running FastAPI applications.
- **sqlalchemy**: The SQL toolkit and Object-Relational Mapping (ORM) system used for database interactions.
- **alembic**: A database migration tool for SQLAlchemy, used to manage database schema changes.
- **pydantic**: A data validation and settings management library used for defining and validating data models.
