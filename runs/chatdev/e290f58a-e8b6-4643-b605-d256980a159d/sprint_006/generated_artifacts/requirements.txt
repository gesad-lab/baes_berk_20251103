
### Explanation of Dependencies:
- **fastapi**: The web framework used for building the API.
- **uvicorn**: The ASGI server used to run the FastAPI application.
- **sqlalchemy**: The SQL toolkit and Object-Relational Mapping (ORM) system used for database interactions.
- **pydantic**: The data validation and settings management library used for defining data models.
