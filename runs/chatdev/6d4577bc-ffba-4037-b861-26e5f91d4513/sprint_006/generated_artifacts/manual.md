# ChatDev Course Management System

Welcome to the ChatDev Course Management System, a powerful API designed to manage students, courses, and teachers effectively. This system allows you to create, read, and manage relationships between students and courses, as well as between courses and teachers.

## Main Functions

The ChatDev Course Management System provides the following main functionalities:

- **Student Management**: Create and manage student records, including their enrollment in courses.
- **Course Management**: Create and manage courses, including assigning teachers to courses.
- **Teacher Management**: Create and manage teacher records.
- **Enrollment**: Enroll students in courses and manage their course participation.

## Quick Install

To set up the environment and install the necessary dependencies, follow these steps:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/your-repo/chatdev-course-management.git
   cd chatdev-course-management
   ```

2. **Create a Virtual Environment** (optional but recommended):
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

3. **Install Dependencies**:
   Create a `requirements.txt` file with the following content:
   ```
   fastapi
   uvicorn
   sqlalchemy
   pydantic
   ```
   Then run:
   ```bash
   pip install -r requirements.txt
   ```

## How to Use the API

Once the environment is set up and the dependencies are installed, you can start using the API. Follow these steps to run the application:

1. **Run the Application**:
   ```bash
   uvicorn main:app --reload
   ```
   This command will start the FastAPI server, and you can access the API at `http://127.0.0.1:8000`.

2. **Explore the API**:
   You can explore the API documentation automatically generated by FastAPI at `http://127.0.0.1:8000/docs`. Here, you can test the endpoints interactively.

### API Endpoints

- **Students**:
  - `POST /students/`: Create a new student.
  - `GET /students/`: Retrieve a list of all students.

- **Courses**:
  - `POST /courses/`: Create a new course (assign a teacher using `teacher_id`).
  - `GET /courses/`: Retrieve a list of all courses.

- **Teachers**:
  - `POST /teachers/`: Create a new teacher.
  - `GET /teachers/`: Retrieve a list of all teachers.

- **Enrollment**:
  - `POST /students/{student_id}/courses/{course_id}`: Enroll a student in a specific course.

## Database Migration

The system includes a migration script to update the database schema. The migration preserves existing data for students, courses, and teachers while adding the necessary relationships. To run the migration, execute the following command in your Python environment:

```bash
python migration_script.py
```

The ChatDev Course Management System is designed to streamline the management of educational entities. With its robust API, you can easily manage students, courses, and teachers, ensuring a smooth educational experience. For further assistance or inquiries, please contact our support team.
