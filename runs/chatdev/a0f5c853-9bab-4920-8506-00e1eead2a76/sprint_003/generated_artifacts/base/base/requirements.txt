
### Explanation of the Dependencies:
- **fastapi**: The main framework used to create the web application.
- **uvicorn**: An ASGI server for running the FastAPI application.
- **sqlalchemy**: The ORM used for database interactions with SQLite.
- **pydantic**: Used for data validation and serialization.
- **requests**: A library for making HTTP requests, used in the GUI for interacting with the FastAPI API.
