
### Explanation of Dependencies:
- **SQLAlchemy**: This is the ORM (Object-Relational Mapping) library used for database interactions.
- **Pydantic**: This library is used for data validation and settings management using Python type annotations.
- **FastAPI**: This is a modern web framework for building APIs with Python, which is often used alongside Pydantic.
- **Uvicorn**: This is an ASGI server that is commonly used to run FastAPI applications.
