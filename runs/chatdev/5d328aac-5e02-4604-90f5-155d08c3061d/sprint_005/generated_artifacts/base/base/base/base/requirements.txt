
### Explanation of the Dependencies:
- **fastapi**: The main framework used to create the web application.
- **uvicorn**: An ASGI server for running the FastAPI application.
- **sqlalchemy**: The ORM used for database interactions and model definitions.
- **pydantic**: Used for data validation and serialization of request and response models.
