
### Explanation of Dependencies:
- **fastapi**: The main framework used to create the web application.
- **uvicorn**: An ASGI server for running FastAPI applications.
- **sqlalchemy**: The ORM used for database interactions with SQLite.
- **pydantic**: Used for data validation and serialization of request and response models.
- **requests**: A library for making HTTP requests, which is used in the Tkinter GUI to communicate with the FastAPI backend.
