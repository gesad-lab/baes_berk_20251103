# Changelog

## [1.0.1] - 2023-10-01
### Added
- Introduced Teacher functionality, allowing for CRUD operations on Teacher resources.
- New API endpoints for managing Teachers have been implemented. This includes the ability to create, retrieve, update, and delete teacher records.

### Changed
- Minor adjustments in response formats to accommodate new Teacher API requirements.

### Fixed
- Resolved issues with existing API error handling mechanisms to ensure consistent responses across new and old endpoints.

## [1.0.0] - 2023-09-01
### Initial Release
- Initial implementation of Enrollment functionality for students and courses.
- Added relevant tests to validate student enrollment processes.
