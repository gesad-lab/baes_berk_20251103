## Setup Instructions

1. **Clone the Repository**
   
   ```bash
   git clone https://github.com/yourusername/your-repo.git
   cd your-repo
   ```

2. **Configure Environment**

   Make sure you have [Poetry](https://python-poetry.org/docs/) installed. Then, create and activate a virtual environment:

   ```bash
   poetry install
   ```

3. **Run the Application**

   You can start the FastAPI application using the following command:

   ```bash
   poetry run uvicorn main:app --reload
   ```

   The application will be running at `http://127.0.0.1:8000`.

## Usage Examples

- To create a new Student record, make a POST request to `/students` with the required JSON body.
- To retrieve all Student records, make a GET request to `/students`.

Example of creating a Student:

```bash
curl -X POST "http://127.0.0.1:8000/students" \
-H "Content-Type: application/json" \
-d '{"name": "John Doe", "age": 20, "class": "Biology"}'
```

Example of retrieving all Students:

```bash
curl -X GET "http://127.0.0.1:8000/students"
```
