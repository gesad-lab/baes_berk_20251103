### II. Technology Stack
- **Backend Framework**: FastAPI (Python) - Chosen for its efficiency and straightforward integration with modern applications.

- **Environment Management**: Poetry - For dependency management.
- **API Documentation**: Automatically generated by FastAPI via OpenAPI.
- **Testing Framework**: pytest - For validating added functionalities ensuring high code quality.

### API Endpoints

#### Create a Course
- **Endpoint**: `POST /courses`
- **Description**: Creates a new course.
- **Request Body**:
  - `title` (string, required): The title of the course.
  - `description` (string, required): A detailed description of the course.
  - `duration` (integer, required): Duration of the course in hours.
  - `instructor` (string, required): The name of the instructor.
  
```json
{
  "title": "Introduction to Programming",
  "description": "Learn the basics of programming using Python.",
  "duration": 40,
  "instructor": "John Doe"
}
```

- **Response**:
  - **201 Created**: Returns the created course object.
  
```json
{
  "id": 1,
  "title": "Introduction to Programming",
  "description": "Learn the basics of programming using Python.",
  "duration": 40,
  "instructor": "John Doe"
}
```

#### Get All Courses
- **Endpoint**: `GET /courses`
- **Description**: Retrieves a list of all courses.
- **Response**:
  - **200 OK**: Returns a list of courses.
  
```json
[
  {
    "id": 1,
    "title": "Introduction to Programming",
    "description": "Learn the basics of programming using Python.",
    "duration": 40,
    "instructor": "John Doe"
  },
  ...
]
```
